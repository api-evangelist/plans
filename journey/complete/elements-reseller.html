{"Elements":"Reseller"}

There are reseller opportunities in place around resources available via the platform, that allow approved API consumers to access resources, as well as resell access to resources. Resellers are paid a fixed amount, or possibly percentages of sales of products and services, which could include accessing and usage of API resources.

Organizations

  • GoDaddy - GoDaddy has a good example of APIs working with a reseller program to allow 3rd parties to earn revenue by putting APIs to work.
  • Google - Google has an interesting reseller API around Google Apps.
  • SendGrid - SendGrid has a reseller API for use as part of deeper integrations by their resellers.
